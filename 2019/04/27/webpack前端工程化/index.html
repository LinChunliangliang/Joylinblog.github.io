webpack

安装webpack的前提是需要安装npm工具或者cnpm工具

Start

指令 相关说明
cnpm view webpack versions 查看webpack可用版本
cnpm view webpack version 查看webpack已安装版本
cnpm i webpack@版本号 -g 安装webpack
cnpm i webpack-cli -g 安装webpack-cli

webpack打包

webpack打包的原理 把多个代码文件内容抽离出来 全部放在打包结果文件中

webpack4打包：webpack 输入文件路径 -o 输出文件路径 –mode 打包模式

main.js中先载入其他js函数库等

webpack ./main.js -o ./bundle.js —mode development

webpack配置

步骤

  1. 先卸载全局下的webpack:cnpm remove webpack webpack-cli -g

  2. 在项目本地安装webpack和webpack-cli:cnpm i webpack webpack-cli -D

  3. 在项目目录下创建webpack配置文件webpack.config.js

  4. 安装jquery:cnpm i jquery -s,在main.js文件中载入jquery模块（详情见12步）

  5. 在webpack的配置文件中，说明输入和输出文件

  1. 在npm配置文件，添加新脚本，使用项目本地安装的webpack软件

  2. 调用npm的脚本npm run dev即可使用,但是每改变代码一次就要重新调用，所以需要安装webpack-dev-server开发服务，类似于nodemon

  3. 安装webpack-dev-server:cnpm i webpack-dev-server -D

    安装好后需要在npm配置文件中把webpack修改为webpack-dev-server

  1. Web-dev-server打包的产出，放在服务的根目录下，其实时一个临时文件，放在内存里，需要修改index.html的引用路径

    <script src="/bundle.js"></script>

  2. web-dev-server详细参数

  1. 主页虚拟化html-webpack-plugin:cnpm i html-webpack-plugin -D

    在webpack.config.js文件中配置导入插件并且配置

  1. webpack打包css

    • 在项目中添加css样式文件index.css

    • 安装合适的载入器cnpm i css-loader style-loader -D

    • 在main.js中导入css文件

    • 在webpack.config.js文件中，说明文件处理规则

  1. webpack打包less

    • less是一门css预处理语言，它增加了变量，mixin，函数等特性，是css更易维护和扩展

    • 安装合适的载入器cnpm i less-loader less -D

    • 在main.js中导入less文件

    • 在webpack.config.js文件中，说明文件处理规则

  1. webpack打包scss/sass

    • 世界上最成熟，最稳定，最强大的专业css扩展语言

    • 安装合适的载入器cnpm i sass-loader node-sass -D

    • 在main.js中导入scss文件

    • 在webpack.config.js文件中，说明文件处理规则

  1. webpack打包图片

    • 安装url-loader载入器，url内部依赖file-loader

    • 安装:cnpm i url-loader file-loader -D

    • 在webpack.config.js文件中，说明文件处理规则

  1. webpack结合babel

    • babel是一个javaScript编译器，babel能把高级语法转化为低级语法 为了让浏览器能够兼容运行

    • 类似于:import $ from "jquery"

    • 总共需要安装5个包（注意事项，babel-loader版本依赖babel-core版本两个版本号只能相邻，前者是8，后者必须是7，前者是7，后者必须是6 ）

    • 安装：cnpmi babel-loader@7.1.5 babel-core -D

      ​ cnpm i babel-plugin-transform-runtime -D

      ​ cnpm i babel-preset-env babel-preset-stage-2 -D

      依次执行以上命令

    • 在webpack.config.js文件中，说明文件处理规则

  2. 如果需要使用到vue，则也需要设置依赖信息

    • 安装:vue：cpm i vue -S

    • 在main.js中模块化导入vue包import Vue from "vue/dist/vue.min"

    • 安装vue依赖:cnpm i vue-loader vue-template-compiler -D

    • 设置处理规则

    • 导入vue模板编译插件

      在webpack.config.js中导入vue插件

      var vlp=require("vue-loader/lib/plugin")

      在plugins里注册插件new vlp

  3. vue-router

    • 安装:cnpm i vue-router -S

    • 在main.js中导入Vue路由模块

  1. 这套配置可以在以后使用，之后创建项目目录后，只要把.babelrc. package.json webpack.config.js三个文件放在项目目录下，然后进行项目复活即可使用
