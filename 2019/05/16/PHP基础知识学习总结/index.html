PHP基础知识点梳理

脚本运行

###PHP脚本的运行方式

  1. CLI模式，在系统的命令中执行PHP脚本
  2. Apache内置模块：通过浏览器访问apache，由apache读取并运行php脚本
  3. FastCGI模式：一般在配置Nginx时使用，能够提高更好的并发性能
  4. PHP内置Web服务器

###MAC版搭建Apache模块运行php脚本

  1. 可选择一个一个软件安装（需要配置环境，有点复杂），所以为了快捷，也可以选择集成环境安装，例如：mamp或者xampp（windows系统可以安装phpStudy）

  2. 因为我是安装了mamp pro，所以我这里介绍一下安装mamp的相关介绍

  3. mamp pro是一个要序列号的软件，如果没有序列号，会有15天的试用期，这里我找到一个破解版：http://www.sdifen.com/mamppro411.html 详情访问网站即可

  4. 安装完成后图标会多一个鲨鱼

  5. 完成后页面

  6. 安装完成后需要进行配置

    1. 第一步

      添加一个新的配置

    2. 第二步

      在Host name里填写你想要通过什么网址访问你的脚本，并且选择你的项目文件目录

    3. 第三步

      在Port里填写80端口

    4. 第四步

      启动服务

    5. 在浏览器通过你设置的网址访问你的脚本，如果是index.php脚本文件，网址后面可以不用写脚本名称，如果不是index.php命名的需要在网址后面加/脚本名称.php即可访问脚本

    6. 配置完成

  7. 后期有新的项目方法一样配置

变量的传值

我们可以把一个变量的值传给另一个变量，在传值时有两种情况：值传递 和 引用传递。

值传递：把一个变量的值在内存中复制一份给另一个变量。（类似于克隆）

引用传递：把一个变量在内存中的地址给另一个变量，两个变量实际上操作同一块内存。（类似于起个别名）

值传递：值传递直接使用 = 即可

引用传递：引用传递时在变量名前使用 & 符号，获取变量在内存中的地址（简单说就是修改一个的值，会改变另外一个的值）
