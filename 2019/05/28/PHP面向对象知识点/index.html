三大特性

面向对象有三大特性：

  1. 封装：常量，变量，函数等代码都要写在类中
  2. 继承（extends）：一个类可以有多个子类，子类会继承父类中的属性和方法
  3. 多态：父类中同一个方法被多个子类继承后，表现出不同的形态

访问控制

可以设置类中常量，属性和方法的访问权限

  • public:公开的，可以随意访问
  • protected:受保护的，在本类和子类中可以访问
  • private:私有的，在本类中可以访问

final

如果一个类不想有子类继承，那么可以在定义这个类时添加final定义不可被继承的类

(同样的，如果类里面的方法不想被继承，也可以在定义方法的前面添加final)

##命名空间

为了区分一些类名的冲突问题，我们可以使用命名空间来区分。命名空间的定义写在php代码中的最前面。

命名空间的定义

当项目比较大时，只用一级命名空间不够时，我们可以使用\定义多级命名空间

如果使用了命名空间，当我们在使用该命名空间中的类时，需要加上命名空间；

如果是在同一个命名空间下，那么可以直接使用类名不用添加命名空间。

引入类（use）

如果要使用其他命名空间中的类，我们必须要在创建类时加上命名空间名；

如果觉得写命名空间比较麻烦，那么我们可以使用 use 先引入类，然后就可以直接使用类而不加命名空间。

总结：

  1. 可以使用use引入一个命名空间下的类
  2. 引入之后，再使用这个类时，不需要写命名空间直接写类名即可
  3. 如果引入两个同名的类，可以使用as起别名

类文件的自动加载

指的是当使用一个类是，自动加载类文件

自动加载函数

PHP 实现类的自动加载有两种方式：

  • __autoload ：当使用一个类时，如果这个类不存在，就调用这个函数（过时）
  • spl_autoload_register：可以用来注册多个类加载函数

spl_autoload_register自动加载函数

魔术方法

魔术方法是指在某个时刻自动被调用的方法

PHP一共有15个魔术方法，其开头都是以__开头

魔术方法 相关解释
__construct() 构造函数（当类被实例化时会调用，经常用于对类初始化）
__destruct() 析构函数（当对象被销毁时会调用，经常用来做一个资源释放的操作）
__clone 当一个对象被克隆时使用这个方法（如果将克隆方法设置为私有的，那么该类将无法被克隆）
__sleep 当一个对象序列化时，这个方法就执行了
__wakeup 当将一个字符串反序列化时，这个方法就执行了
__get 当获取一个类中不存在的属性时，它就调用了
__set 当修改一个对象中不存在的属性时，它就调用了
__toString 当你把一个对象当字符串直接输出时调用
__call 当我们调用一个对象中不存在的方法时
__callStatic 当我们调用一个对象中不存在的静态方法时
__invoke 当你把一个对象当做函数来调用时，这个方法被调用了

​ 持久化：把数据长久的保存起来，把数据写到硬盘、写到数据、写到文件里、USB等媒体上

​ 在 PHP 中只有数字、字符串类型可以直接持久化，数组、对象等复杂的数据类型无法直接持久化。

​ 所以，如果我们希望将对象、数组进行持久化，那么我们首先要将对象先 序列化 ，序列化之后就得到了一个字符串，然后就可以将这个字符串持久化。以后要用时，可以再将这个字符串 反序列化 回对象使用。

序列化：将一个对象根据一个算法转成一个字符串，使用 serialize 函数。

反序列化：将字符串根据算法转回对象，返回 unserialize 函数。
